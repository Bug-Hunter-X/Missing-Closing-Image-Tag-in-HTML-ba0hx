The solution is simple: Add the missing closing tag to the <img> tag. Here's the corrected HTML:

<div>
  <p>Some text</p>
  <img src="image.jpg" alt="Image" />
</div>

Note the addition of />. This correctly closes the <img> tag, ensuring proper rendering and preventing potential layout issues. While some browsers may tolerate missing closing tags for self-closing elements like <img>, it's always best practice to include them for consistent behavior across all browsers and to make your code more maintainable and readable.
